# occquery_v0 -- results summary

occquery is a deterministic instrument that **MEASURES box-blind static occupancy geometry**
(free-width, clearance, free-path) -- physical quantities a box-only query language cannot express.
It does NOT judge danger: whether a measured situation is dangerous (a lead car vs a wall, a near-miss
vs a parked car) is relative motion over time = the **dynfield** experiment (PLAN s4 Scope). occquery
is judged on MEASUREMENT accuracy + expressivity, never on retrieving "danger".

Result classes, kept distinct: unit/integration tests; synthetic smoke (`smoke.md`, generated by
`run.py`); real-data measurement on Occ3D-nuScenes mini. An externally validated denotation
(measurement accuracy vs an independent oracle) is NOT done yet.

## H1 -- expressivity (oracle-free, the headline) -- HOLDS

Box-blind witness (`tests/test_expressivity.py`): two scenes with identical tracked-box observables
differing only by an unboxed occupancy wall -- the box backend MUST return the same answer, the
occupancy predicate distinguishes them. RefAV's released 33-function set has 0 free-space primitives
(`docs/expressivity-vs-refav.md`). Checkable by anyone, no oracle.

## Real Occ3D-nuScenes mini -- what occquery MEASURES

v1 predicates measure free-width / clearance / free-path over a C-space reachable field
(`src/probe/predicates/reachable.py`). The earlier "v0 0/5 precision" framing was a **category
error**: a static measurement graded against a dynamic-danger answer key. Walking scene-0061 in the
3D viewer showed its 0.80 m free-width is a CORRECT measurement of the gap beside a **lead car** (the
ego follows it, decelerating 5.1 -> 2.2 m/s over 36 m) -- not a false positive. The danger verdict
(lead car vs wall) belongs to dynfield. Full account + viewer walk-through:
`docs/h3-real-data-findings.md`.

## NOT done (honest)

- **H3 independent oracle**: measurement accuracy (free-width / clearance) vs a LiDAR-derived
  geometric oracle, released for recompute.
- **H2**: RefAV / EvalAI HOTA-Temporal leaderboard.
- **danger judgment** (lead car vs wall, near-miss vs parked) = the dynfield experiment.
- **val split** (mini only); **sub-0.4 m clearance** (voxel-resolution bound -- needs raw-LiDAR
  re-voxelization).

## Decision: PURSUE

H1 (expressivity) is strong and oracle-free; the predicates MEASURE box-blind geometry, and v1 made
that measurement honest (C-space reachability, noise-robust). Next: H3 independent oracle
(measurement accuracy), then the dynfield experiment for the danger / dynamics layer.
